      巴别塔

      date: 2026-02-11
      edited: 2026-02-12 11:00

      s1#

      (乡村 - 晴)

      大家热闹的造房子, 拿起木头, 怎么高就怎么造

      旁观者: 不该建房，地基不对

      狗帮助叼木材, 对旁观者汪汪叫

      s2#

      (日夜变换)

      奶奶一层盖出一层，新层放父亲

      奶奶 (楼下嘲讽) 年纪大了还单身, 长大娶了老娘没人敢进来

      (黑幕一会)

      父亲和母亲把孩子放在了3楼

      奶奶: (只出现声音. 一楼骂. 突然很惊讶的啊了出来. ) 这是亲戚们送的礼啊, 过几天要送回去的, 这个小孩长大也是作孽, 拆礼物到村里老公面前都没人忍, 踢出家门又回来呜呜哭, 说自己怀孕了, 更加没人要.

      s3#

      (晚, 楼下)

      旁观者：孩子都那么大了，停下吧。

      (晚, 三楼)

      孩子晚上醒来

      自己的床微微摇晃

      楼下母亲的声音传上来

      母亲(只出现声音 轻声) : 亲戚送礼又马上要回礼, 一模一样的礼物送来送去. 小孩子不懂事拆个礼物都骂那么难听,你妈, 还有你爸… 全家都死了好了

      **父亲 (**闷哼)

      孩子的床晃动的原来越大, 伴随咯吱咯吱的声音

      (黑幕一会)

      孩子依旧晚上睁着眼

      楼下两人咚咚的打架推墙

      (闷声嘶叫, 小声骂人, 墙的声音也是闷的)

      她看到墙上的裂缝慢慢变大蔓延

      声音和震动突然停止了

      (晚, 二楼, 无灯)

      父亲蹲在一旁抱头, 母亲的太阳穴卡在床脚, 一动不动。

      (黑幕一会, 一年后, 明亮的灯)

      父亲带着新女友开心的进卧室

      卧室多了个壁炉.

      s4#

      (早, 三楼)

      父亲敲门, 孩子开门

      父亲(拿着新衣服在门外): 你几天没洗澡了?

      孩子: 两三天

      父亲: 要一天洗一次澡, 头最多两天就要洗了.

      父亲: 奶奶腿疼上不来楼, 天天要我给你新衣服穿. 不然别人会笑话我们穷的.

      父亲: (打开女儿的衣柜, 拿起深色系的衣服) 这件衣服你买的? 多少钱?

      孩子: 几十块

      父亲: (哑声笑) 你看这材质, 一拉就坏了, 不要了?

      孩子: 随便你

      父亲: (继续翻衣柜, 双手抓起内裤检视) 内裤下次阿姨给你买好一点的, 这内裤都有洞了. 不要了?

      孩子: 随便

      父亲: (站起来收拾东西) 住在这里要保持干净, 你看这里还有落地窗呢. 以后你带孩子回来住, 外面景色清爽, 房间干净, 大家心理也舒服, 不是吗?

      (父亲翻完衣柜起来, 把孩子的机械键盘, 画架, 家具拆掉, 面包都用水泥糊墙裂缝当支撑.

      屋里只留下了巴洛克式的大床和大衣柜)

      父亲: 这样, 多清爽.

      (狗在门口晃尾巴, 对孩子笑)

      父亲(转头对狗): 下午有批木头到了, 记得搬下.

      s5#

      (晚, 二楼, 有灯)

      父亲: 她这个月没要卫生巾, 储备也不多了才对.

      父亲女友: 带她去医院检查下吧

      父亲:(对女友笑) 怀孕了?

      父亲女友:(严肃的说) 月经不来, 以后影响孩子了该怎么办?

      (画面从二楼过度到三楼)

      (晚, 三楼, 无灯)

      楼下的床有咯吱咯吱的声音, 孩子的床依旧在不由自主的晃.

      孩子屁股翘着趴着在床上, 有狗在她身后抽搐

      和狗的性交比人的安静很多

      s6#

      (早, 医院 )

      医生 从b超检测门出来, 递给了父亲检测单

      检测单: 子宫缺失

      父亲(表情凝重)

      (下午, 家, 三楼)

      孩子 (走回房间)

      父亲 跟在后面看着她回房间, 等她快进房间 发怒, 猛的跑过来掐住她的双肩, 瞪着她, 拳头打穿墙.

      然后立马把孩子抓到了三楼屋顶, 让她站在那里

      怒气冲冲的和水泥. 把她当罗马柱

      一开始很粗暴的她身上丢水泥

      往复平抹抚平了他焦躁的内心

      完工之后, 他找来女儿的画笔, 勾勒出罗马柱的花纹和线条

      不知不觉到了晚上, 奶奶喊吃饭, 他下去了.

      孩子听没动静, 乘着水泥没干跑掉了, 屋顶颤颤悠悠, 她掰开了屋子的裂缝跳到了3楼, 2楼, 1楼, 跑掉了.

      (晚, 家, 二楼)

      晚上和女友上床, 咯吱咯吱声又重新响起

      在加速高潮的过程中, 房子倒塌

      村里狗叫, 交配.

      父亲倒在石砖和占了水泥的废料里,

      爬出来的手拍到了狗屎上, 在石砖上擦狗屎,

      石砖下有块布, 于是开心的擦了起来

      有多余的液体. 月光下是黑黑的, 抬手看是血

      擦屎的布是奶奶的衣服.

      奶奶被压死了, 女友被压死了.

      父亲赤裸着面对全村人的疑惑

      惊恐的找寻着中间有骨头的水泥, 藏到了身下.

      s7#

      (晚, 林间)

      旁观者 (看着倒塌的景观拉屎)

      (看到了身下的背影, 转过头去)

      **孩子(**递来一张家庭建筑草图的纸，纸上的塔像空中花园)：“他们花三天设计的原稿。” 旁观者：“至少还有个长处” **旁观者 (**擦完，丢到屎里, 站起来. 踢土把一起埋了) 孩子：“路上全是建筑，林子被砍得就剩这么点了” 旁观者（下巴点了点孩子远处的家）：“最后都会变成那样”

      孩子：“烂得比我们更快”
