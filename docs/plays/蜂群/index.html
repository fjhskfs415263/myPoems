      蜂群

      蜂群

      date: 2025-10-05
      edited: 2025-10-05 16:14

      孩子的眼睛总能发现一个世界。这天他跑出了家来到树林。蜂巢选出了新蜂后。
      旁白: 没人知道 queen 被选中前叫什么。但选中了后，它就是 queen。

      时间: 室内, 下午, 晴
      Queen 小时候:(体态和其他蜜蜂一样,坐在雄蜂腿上看着窗外. )

      Queen: “你们天天飞在外面都在干什么? 我从来不被让出去.”
      雄蜂 (神经质的抽搐, 把手里的精液捧到 queen 嘴边): “吃吧, 这很有营养的”
      Queen (凑近闻): “臭臭的. 有点像母亲身上的味道.”

      时间: 室内, 清晨, 晴
      Queen 成长期 (和雄蜂睡在一个床上, 但明显比雄蜂大了一圈. 刚醒):

      雄蜂(贴近看着 queen 醒来): “你好美.”
      Queen: “你什么时候醒的? 今天你要飞出去玩吗? "
      雄蜂: “我昨晚把手插进你里面, 你没反应. 但某刻你动了, 我很害怕. 但你只是在做梦.”

      时间: 第一天: 室外. 集会, 下午:
      Queen 快成熟期 (体型臃肿, 脚被体重压弯了难以移动. 翅膀很微小, 说话吃力)

      (工蜂们骚乱, 嗡嗡声乱糟糟的)
      雄蜂: “好了! 安静各位! 今天是特别的日子. 我们的 queen 举行成人礼.”
      (工蜂们在演讲时安静了一会, 后爆发出笑声, 还有连绵的口哨声 )
      雄蜂 (笑): “作为 queen 半个的童年监护人, 今天由我来为这神圣的使命献身. "
      (雄蜂们的笑声更猛烈了)
      (雄蜂走到 queen 的身后, 猛地抱住 queen 的后段躯体. 拿硬化的下体对准 queen 的屁股缝摩擦. Queen 没反应, 只是拼命震动她的小翅膀, 发出蚊子一样的嗡嗡声
      在生殖器对接前, 雄蜂暗暗咬了她一口. Queen 惊叫了一声, 本来慢慢沉默下来的工蜂们再次沸腾了起来 )
      雄蜂(在 queen 的后尾愤怒地抽搐): “这是我最后一天了！你居然还和僵尸一样？死了吗？ 被爸爸操得爽不爽? 嗯？既然你和别的蜂私交得挺爽的，被公开操一定爽死了吧？ "
      (queen 动了动后尾, 让雄蜂抽动得更快了, 雄蜂高潮后四肢僵直。怔住了不敢拔出来。 queen 倒是抽动得更厉害了,一下子把排泄物喷射出来)
      (雄蜂猛地抽出身来，惊吓让他忘记了疼痛。工蜂们爆笑。 雄蜂自己下体的伤口也被褐色的排泄物污染, 恐慌屈辱让他尴尬无措的离去)

      时间: 第一天：室内. queen 寝室, 晚上.

      雄蜂 (开 queen 的门): “恭喜你明天就要成为真正的 queen 了. 你飞行练得挺不错嘛”
      Queen (吃力的哼了一个低音, 代表友善的回音, 继续拍她的翅膀, 让人厌烦的蚊子嗡嗡声再度袭来)
      雄蜂(把脸靠在queen面部区域, 沉默许久): “这是我最后一次见你了.”
      雄蜂 (擦了擦眼泪): " 哈哈, 以后你活成了寡妇该怎么办啊? "
      雄蜂 (绕到 queen 的后尾): " 我给你带了个礼物”
      (雄蜂把触手放在 queen 的阴道里摩擦, 累了就停止了. queen 发出均匀的咕噜声, 她已经睡去了. )
      (雄蜂把今天抓来的水蛭放进 queen 的阴道里, 离去了)

      时间: 第二天：室内. Queen 寝室, 早上

      Queen (醒来发现翅膀没有重量了, 无论怎么样都发出不了嗡嗡声了, 在吃力转身, 碰倒椅子, 衣架, 水晶吊灯被她弄得哗啦哗啦响. 终于转到了梳妆台的位置, 挤压身体脂肪, 看到她背上的翅膀已经没有了.)
      Queen (到了该排卵的时间, 拉出来透明的东西. 她自己越来越容易饥饿. 她学会了拉一段时间的卵, 然后掉头, 刚好可以吃到卵. 卵臭臭的, 和母亲和雄蜂身上的味道一样. )
      工蜂(两腿各腿各一球花粉, 边走边洒花粉. 进了 queen 的房间, 看到这一幕, 震惊): “我们天天给你准备那么多吃的, 这就是你的产出吗?”
      Queen (和没听到一样, 继续吃. )
      (工蜂见 queen 要转身, 赶紧跑了出去)

      时间: 第二天：室外. 集会, 早上
      工蜂 (疲惫大叫): “queen 居然在吃自己的卵! "
      其他工蜂 (放下手里的活, 很诧异) : “但她不是 queen 吗? "
      工蜂 (厌烦地): “跟我去看你就知道了!”

      时间: 第二天：室内. Queen 寝室, 早上
      (工蜂们恐慌的看着 queen 吞噬卵) : “她到底在干什么啊? 我们的18 年的辛苦劳动就是养这个畜生的吗? 吃得那么胖, 什么都不干? 现在还吃孩子? "
      工蜂(颤抖的对 queen 大喊): “嘿! 你知道你在干什么吗!? "
      (Queen 好像听到声音, 放下手里的卵想凑过去听, 但大家仓皇而逃.)

      (跑出不远的地方)
      工蜂 (气喘吁吁的): “真是个畜生.”
      工蜂们(同样气喘吁吁, 恐慌地): “我们能怎么办呢? "
      工蜂: “团结力量大”

      时间: 第二天：室内. Queen 寝室, 中午
      (群体密密麻麻的工蜂和雄蜂们挤到了 queen的寝室, 团结起来掰断了 queen 的头)

      • （queen直到头被掰下，才看到了掉在地上的项链。挂坠是两片蜜蜂的翅膀。）

      • （queen嘴唇异常的轻松，快速的撅起又放下，像是放下了18年的重负）：“free…！ free…！”

      • (大家在等 queen 的死亡, 但是 queen 的无头躯体像活着的体态–阴道孩子继续漏卵, 身体还在正常蠕动 )
        (大家屏息看着里面好像有东西要出来: 水蛭从无头尸体里慢慢探出头, 这个水蛭已经和 queen 的身体差不多大了. 大家被吓得仓皇而逃.)

        时间: 第三天, 集会, 中午.

        旁白: 社区一片安宁, 蜜蜂群体里又选出了新 queen. 没人知道 queen 被选中前叫什么。但选中了后，它就是 queen。
        男孩 (嘻嘻笑着收起了自己的戏剧腔), 但想到: “这么多时候了，研究蜂群的人以为蜂群中有独特的语言。因为整天的嗡鸣声, 还有各种奇特的舞蹈。”
        (他看着工蜂们团结一致的拆解 queen，做储备粮, 选出下一个 queen。)
        男孩继续想: “但做出新选择前只是带头的做出选择，其他只是跟随。也许大家都只是昆虫，大多只是在不损害核心利益的条件下模仿。其实谁也听不懂谁。”

        时间: 第三天, 男孩家, 中午

        (男孩跑回家, 兴奋的把这一发现告诉了沙发上看报纸的父亲.)
        父亲 (摸摸他的头): “你这么热爱自然很好, 但是专家们几百年的科学研究一定有他们的道理. 要努力读书, 长大挣钱，读书的目的就是为了挣钱。才能找个好老婆. 现在去做作业吧，以后别做这些有的没的了. "
