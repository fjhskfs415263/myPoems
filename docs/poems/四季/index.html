    四季

    四季

    date: 0001-01-01
    edited:

    9.20.25. 1:52

    我站在门口
    学生们结伴穿梭
    人迹点亮小路
    又一个夏天结束了

    秋天抽出金黄的芽
    地里钻出的果实已
    不再让农民喜悦。
    某天学生们会换上厚袄
    接着是新年
    彩灯抵御
    无边的沉寂

    寒冷钻进被褥
    一些人在
    鞭炮声中咽气
    举杯的人
    欢庆又一年的新生

    春天，大地的孩子
    欢笑着跑满人间
    跨过苍白的骨头
    把祝福洒满世界
